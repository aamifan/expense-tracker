# expense-tracker
# 简易记账工具

一款简单易用的记账工具，帮助您记录日常开销，支持数据可视化和本地数据存储。

## 功能特点

- 💰 **快速记账**：随时记录每笔开销，包含金额、类别、日期和备注
- 📊 **数据可视化**：通过饼图展示支出分布，折线图展示每日支出趋势
- 📱 **响应式设计**：完美适配手机和电脑等各种设备
- 💾 **数据持久化**：使用浏览器本地存储，确保数据不会丢失
- 🚀 **PWA支持**：可作为类原生应用添加到手机主屏幕

## 如何使用

### 在线使用

1. 访问 [简易记账工具](https://your-username.github.io/expense-tracker)（替换为您的GitHub Pages链接）
2. 在手机浏览器中，可选择"添加到主屏幕"，获得类原生应用体验

### 本地开发

1. 克隆本仓库
    ```bash
    git clone https://github.com/your-username/expense-tracker.git
    ```
2. 打开 `index.html` 文件即可运行

## 部署到GitHub Pages

1. Fork本仓库到您的GitHub账号
2. 启用GitHub Pages：
    - 进入仓库设置
    - 找到"Pages"选项
    - 选择"main"分支作为源
    - 点击"Save"保存
3. 等待自动部署完成（通常需要1-2分钟）
4. 访问生成的GitHub Pages链接

## 技术栈

- HTML5 + CSS3 + JavaScript
- Tailwind CSS：用于快速构建响应式界面
- Chart.js：用于数据可视化
- Font Awesome：提供丰富的图标资源
